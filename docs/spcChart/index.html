Allow user to enter an array of number values via a text field. Also to select the type of chart to show, from Standard-Deviations to Percentiles. And allow an editable label for the chart. Finally, create a link with the values, type and label encoded in the URL, and use a stand-alone version of the chart to display the results. That makes a shareable link.

A third variation of the standard-deviation chart, that can take non-normal distributions. It displays points representing a two or three point moving average, instead of the raw data. This guarantees a rough normal distribution.
