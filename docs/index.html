Sudoku Helper Tool

This is an older tool, created to be ES2015-compliant, but recently updated. Fairly self-explanatory, it has a "working" view that visualises possible values for a given cell based on existing entries.

Comparison Tools

When running a series of A-B tests, I found that it took a long time to get enough data points to reach a statistically significant conclusion. I wanted a way to see what the tests were signalling even with a small data-set. These apps represent a way to visualise the results of two or more tests, to see if there really is any difference between them.

The first tool creates bell-curves based on ratios of trials:wins. It is useful to see how much the curves overlap, giving an intuition for where the data might be heading.

The comparitive box-plot tool uses box and whisker plots to display different (small) data sets on the same axis. The idea is to make differences more apparent even with a small set of numbers.

Chart Editing Tools

At one point during a project, I needed a way to make diagrams of code designs using simple text-based descriptions. I wanted to include the diagram definition in source code comments, while also auto-creating diagrams.

This tool uses a very simple textual definition of a state machine based on the Sketch Systems input format. It then converts the definition into the DOT format, suitable for use in GraphViz for rendering as an image.

This tool does a similar thing, but for a hierarchical structure. Representation is simply by nested indents, similar to YAML formats.

Linear Equations Tool

I created this simple tool for using row reduction to solve systems of linear equations. The tool uses basic row operations on a matrix, which makes rearranging the equations much simpler.

Colour Extrapolation Tool

I created this simple tool as a way to explore simple linear colour extrapolation. Turns out colour extrapolation is a complicated subject. However, the tool can be interesting to play with as a way of checking your own colour-sensitivity, by seeing how well you can distinguish between the different adjacent colours.

And this version of the tool uses the HSL colour coding, which gives much nicer results. I also used simple roll-your-own components to create the slider controls. This made detecting updates much simpler, so the tool is nicely performant.
